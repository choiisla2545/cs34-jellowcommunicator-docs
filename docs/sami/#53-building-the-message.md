## Building the message for Jellow
1. Select a category
- Tap a category button, such as food, people, actions, or feelings.
2. Choose an Icon
- Tap an Icon to represent what you want to say
3. Check the Sentence bar.
- Confirm selected icon appears
4. Add more Icons.
- add until your message is complete
5. Review your message
6. Tap Speak
- select the Speak button to hear the completed message.
