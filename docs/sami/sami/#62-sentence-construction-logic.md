## Typing a Message
1. Open **Jellow** app
2. The user taps icons or types words using the keyboard.
4. The selected words or icons are added to the sentence bar.
5. The words are placed in the same order they were chosen.
6. Jellow changes the icons into written text.
7. The text is combined to create a complete message.
8. The message is then sent to the text-to-speech feature, which reads it aloud.
