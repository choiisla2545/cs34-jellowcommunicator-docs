## Building a Communication Board in Jellow
1. Open Jellow and go to the board management section.
2. Select Create New Board.
3. Enter a name for the board.
4. Choose a layout for your board.
5. Add categories or folders if needed (such as Food, People, or Activities).
6. Add icons, pictures, or words to the board.
7. Edit each button by adding a label, image, or spoken message.
8. Arrange the buttons in a way that is easy to find and use.
9. Save your changes.
10. Preview the board to make sure everything works correctly.
11. Make any needed adjustments.
12. Start using the board to communicate.
