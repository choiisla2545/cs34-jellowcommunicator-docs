## Input Mechanism

The Jellow Communicator interface uses multiple input methods to allow users to communicate effectively.

---

### 1. Touch Input
- Users tap icons or buttons on the screen
- Each icon represents a word, phrase, or concept
- Tapping multiple icons builds a message

---

### 2. Category Selection
- Users tap a category (e.g., Food, Feelings, Actions)
- Related icons appear within that category
- This helps organize communication options

---

### 3. Expressive Buttons
- Additional buttons allow users to express emotions or responses
- Examples include:
  - Yes / No
  - Happy / Sad
  - Needs / Wants

---

### 4. Keyboard Input (Optional)
- Users can type custom messages using an on-screen keyboard
- Typed text can be converted into speech output

---

### 5. Gesture or Navigation Support (if enabled)
- Some versions may support accessibility gestures
- Users may navigate using assistive tools depending on device settings
