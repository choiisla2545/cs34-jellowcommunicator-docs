# Jellow Communicator User Guide  
## Launching the Interface

This guide explains how to launch and begin using the *Jellow Communicator* interface.

---

## Prerequisites

Before launching the app, make sure:

- Jellow Communicator is installed on the device  
- The device is powered on  
- Accessibility and language settings are properly configured  
- Volume is turned up for speech output  

---

## Step 1: Open the Application

1. Locate the *Jellow Communicator* icon on your device.
2. Tap or click the icon.
3. Wait for the application to load.

*Expected Result:*  
The Jellow home screen appears.

---

## Step 2: Access the Main Interface

1. Observe the communication board.
2. Identify key elements:
   - Category buttons  
   - Expressive buttons  
   - Settings menu  
   - Optional keyboard feature  

*Expected Result:*  
The main interface with communication tools becomes visible.

---

## Step 3: Select a Communication Category

1. Tap a category (e.g., Food, People, Activities).
2. Browse available icons inside the category.

*Expected Result:*  
Relevant communication icons are displayed.

---

## Step 4: Create a Message

1. Select an icon representing your message.
2. Use expressive buttons if needed.
3. Listen to the spoken output.

*Expected Result:*  
The app speaks the selected message aloud.

---

## Step 5: Access Additional Features (Optional)

- Adjust settings such as:
  - Voice volume  
  - Speech speed  
  - Voice type  
  - Language  

- Use extra tools:
  - Custom boards  
  - Keyboard input for text-to-speech  

---

## Troubleshooting

| Issue | Solution |
|------|----------|
| App does not open | Restart device and try again |
| No sound output | Check volume and speech settings |
| Missing icons | Verify selected category |
| Wrong language | Change language in settings |

---

## Completion

The interface is successfully launched when:

- The home screen is visible  
- Categories and icons can be selected  
- Speech output works correctly
